Welcome to the Home Network Traffic Anomaly Detection System

This system is designed to ensure the security and reliability of your home network. It uses a trained machine learning model to monitor network traffic in real-time and detect anomalies or potential attacks.

As soon as an anomaly is detected, the system will automatically send an email notification to the user. The email will include detailed information about the detected attack and specific instructions on how to respond to the threat effectively.

Explore the dashboard to view key metrics such as total network packets, protocol distribution, and details about detected anomalies. Use the email settings to customize notifications and stay informed about the security status of your home network.
